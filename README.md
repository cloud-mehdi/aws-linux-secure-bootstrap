
 aws-linux-secure-bootstrap
=======
 AWS Linux Secure Bootstrap

This project documents a complete, secure bootstrap of a Linux server on AWS
from zero to production-ready SSH access.

# What this project demonstrates
- EC2 instance creation on AWS
- Secure SSH access using key-based authentication
- SSH alias configuration on macOS
- Linux server access validation
- Clean project documentation structure

# Project Structure
- phases/      Step-by-step documentation of each phase
- screenshots/ Visual proof of configuration (no secrets)

# Why this project matters
This project shows practical cloud and Linux administration skills,
focusing on security, clarity, and real-world workflows.
